## 작업 내역

### 1. 요구사항 정의
- 58개 입찰공고 사이트를 한 곳에서 검색하고 원본 사이트로 이동하는 포탈서비스 기획
- 데이터 소스: 엑셀 파일 (`●입찰 공고(사업 공고) 기관별 입찰 정리.xlsx`)
  - Sheet1: 58개 기관 (기관명, 구분, 카테고리, URL)
  - Sheet2: 나라장터 검색 키워드 85개 (5개 그룹)

### 2. 구현 방식 검토
- **방법 1**: 링크 포탈 (각 사이트 바로가기) → 즉시 가능
- **방법 2**: 크롤링 통합 (자동 수집) → 가능하지만 유지보수 비용 큼
- **결론**: API 우선 + 링크 포탈 병행 방식 채택
